# Frontend requirements (LibraryAI)

This frontend is a static (no-build) HTML/CSS/JavaScript app.

## Runtime requirements

- A modern browser (Chrome/Edge/Firefox)

## Local development (recommended)

To avoid CORS/file:// quirks and to match real HTTP behavior, serve the frontend over HTTP.

Option A (Python):
- Python 3.x
- Run: `python -m http.server 5500`

Option B (Node):
- Node.js 18+ (or any recent LTS)
- Run: `npx http-server -p 5500`

## External services

- The frontend depends on the Backend API being available (default: http://127.0.0.1:8000)
- Configure the API base URL via `API_BASE_URL` in the portal scripts
